# QuietSleep Mobile (Expo)

Expo-based iOS/Android app for QuietSleep.

## Run
```bash
cd apps/mobile
npm install
npm run start
```

### Web preview
```bash
npm run web
```

## Push notifications (Expo)
- Uses `expo-notifications`.
- Tap **Enable notifications** to request permissions and get the Expo push token.
- For production, wire the token to your backend for sending pushes.

## Notes
- Designed with a soft neutral palette (no blue/purple gradients).
- Default language: English.
