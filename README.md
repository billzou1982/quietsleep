# QuietSleep

A minimal, gentle sleep-aid web app built with **Next.js + TypeScript + Tailwind**.

## Features
- Sleep guide (breathing/relaxation)
- White & pink noise generator
- Sleep timer (auto stop)
- Bilingual UI (中文 / EN)
- Local preferences storage

## Getting Started
```bash
npm install
npm run dev
```

Open <http://localhost:3000>.
